Features

Everything you need for meaningful diffs.

AST-aware diff

Understand semantic changes like moved functions and renamed variables.

Side-by-side view

Compare files with aligned lines. Spot differences at a glance.

Inline editing

Accept left, accept right, or edit manually—all without leaving vex.

Blazing fast

Git integration

Pipe git diff output directly to vex for enhanced visualization.

Install

Get vex in minutes. Requires Zig 0.13.0+.

git clone https://github.com/ydah/vex.git
cd vex
zig build -Doptimize=ReleaseFast
sudo cp zig-out/bin/vex /usr/local/bin/

Git integration

git config --global diff.tool vex
git config --global difftool.vex.cmd 'vex "$LOCAL" "$REMOTE"'

Platform support

Linux (x86_64, aarch64), macOS (Intel, Apple Silicon). Windows planned.

Usage

Simple commands, powerful output.

Basic

vex old.zig new.zig
vex -s old.zig new.zig
vex -u old.zig new.zig

AST mode

vex --ast old.py new.py

Git pipe

git diff | vex -
git show HEAD | vex -
